Django CRM
API Documentation
Django CRM provides a comprehensive REST API. All endpoints follow RESTful conventions and return JSON responses.
Authentication
JWT-based authentication with access and refresh tokens. Login at /api/v1/auth/login/ to get your tokens.
Organisations & People
Full CRUD for organisations and people. Endpoints at /api/v1/organisations/ and /api/v1/people/.
Team-based Access
All CRM data is scoped to teams. Users can only access data belonging to their current team.
